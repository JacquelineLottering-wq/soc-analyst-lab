# SOC Analyst Lab – Threat Detection & Log Analysis

## 🧠 Project Overview
This project simulates the responsibilities of a Security Operations Center (SOC) Analyst by analysing security logs, investigating alerts, and identifying potential threats using a SIEM environment.

The goal of this lab is to demonstrate practical SOC analyst skills, including log analysis, alert triage, and incident investigation, using real-world scenarios.

---

## 🎯 Objectives
- Analyse security logs for suspicious activity
- Investigate alerts generated by a SIEM
- Distinguish between false positives and real threats
- Document findings clearly and professionally
- Practice SOC analyst decision-making

---

## 🛠 Tools & Technologies
- Elastic SIEM
- Windows Event Logs / Sample Security Logs
- Linux (Ubuntu)
- VirtualBox
- Wireshark (supporting analysis)

---

## 🔍 Lab Scope
In this lab, I perform:
- Log ingestion and review
- Alert investigation and triage
- Identification of abnormal behaviour
- Basic incident response analysis
- Documentation of findings

This lab focuses on **blue team** detection and response activities.
