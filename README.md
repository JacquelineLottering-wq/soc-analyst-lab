# SOC Analyst Lab Portfolio

This repository demonstrates practical Security Operations Center (SOC) investigation and incident response skills aligned with Tier 1 / Junior SOC Analyst responsibilities.

The scenarios simulate realistic security alerts across identity, endpoint, network, and email threat categories.

## Investigation Areas
- Authentication attack detection
- Suspicious script execution
- Malware detection and containment
- Network threat analysis
- Phishing incident response

## Skills Demonstrated
- Security alert investigation and triage
- Log and telemetry analysis
- Threat behaviour assessment
- Incident response procedures
- Severity classification and documentation

## Repository Structure
alert-analysis/ – Individual alert investigations  
incident-response/ – Response procedures and playbooks  
screenshots/ – Supporting visuals and evidence
